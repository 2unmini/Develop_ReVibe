# 📑 서비스 소개

## 📑 목적 및 필요성

한정판에 열광하는 MZ(밀레니얼+Z세대)를 잡기 위해 기업들은 한정판 제품 수를 크게 늘리고 있습니다. MZ세대들은 개성있는 제품, 희소성 있는 제품이라면 큰 관심을 보이기 때문입니다.

스니커즈 문화가 단순한 패션을 넘어 하나의 투자 자산이자 라이프 스타일로 자리 잡았습니다. 그러나 리미티드의 스니커즈를 구매하거나 판매하는 과정은 여전히 쉽지 않습니다. 
정가 구매는 어려운 반면, 리셀 시장은 불투명하고 가격 변동이 심하죠. 우리는 이런 문제를 해결하기 위해 공정하고 투명한 한정판 스니커즈 입찰 거래 플랫폼을 제공합니다.

ReVibe의 리커머스 서비스를 사용해보는 건 어떠세요 ? 
<br> 구매하고 싶은 물건을 검색하고, 원하는 가격에 거래를 시도해보세요 !

## 📑 **문제 정의와 솔루션**

보통 중고 거래는 신상품보다 저렴하게 사고파는 것이 일반적인 상식입니다. 하지만 리셀 시장은 희소가치가 높은 상품에 프리미엄을 얹어 되파는 형태로 거래되기 떄문에 신상품보다도 가격이 오히려 더 비싸는 경우가 많습니다.
ReVibe에서는 판매자의 원하는 가격에, 구매자의 원하는 가격에 거래가 안정적으로 이루어집니다.

- **Problems**
    - 대중적인 스니커즈 브랜드와 패션 브랜드 또는 아티스트가 협업한 상품은 수요를 공급이 따라가지 못해 소장가치가 높다.
  
- **Solution**
    - 입찰 거래 시스템 도입 : ReVibe에서는 구매자 혹은 판매자로서 참여할 수 있습니다. 입찰, 판매 시스템을 통해 구매자와 판매자 간의 합리적인 거래가 성사될 수 있게 돕습니다.

## **📑** 사용자는 누구인가?

- **Target**
    - 남들과 다른 신발을 신거나 수집하고자 하는 구매자
    - 한정판 제품을 정가에 구매한 후 비싸게 되파는 리셀러

## 📑 핵심 기능

| 기능           | 설명                                        |
|--------------|-------------------------------------------|
| 일반 / 소셜 로그인  | 사용자는 일반 또는 카카오 소셜 로그인을 한다.                |
| 브랜드와 상품 등록   | 관리자는 상품의 세부 사항(브랜드, 상품, 옵션 등)을 등록한다.      |
| 판매 입찰        | 판매자는 원하는 상품의 옵션을 선택하여 판매 입찰을 등록한다.        |
| 구매 입찰        | 구매자는 상품에 대해 원하는 가격 또는 즉시 구매 입찰을 신청할 수 있다. |
| 체결           | 입찰 시 구매 입찰과 판매 입찰의 가격이 일치할 경우 성립된다.       |
| 좋아요          | 로그인한 사용자는 특정 상품을 관심 목록에 추가할 수 있다.         |
| 결제           | 포인트와 쿠폰 사용으로 상품 결제를 할 수 있다.               |
| 리뷰 작성        | 포토 리뷰 시, 500 point, 글 리뷰 시, 100point 부여   |
| 선착순 쿠폰       | 관리자가 발행한 쿠폰을 사용자들이 발급 받을 수 있다.            |
| 사용자 간 1대1 채팅 | 이메일로 상대방을 초대하여 실시간으로 채팅을 할 수 있다. |
